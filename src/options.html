KeyPunk Options

General

Shows generated password in popup. This may also be toggled directly from the popup.

Stores the master password in memory for the duration of the browser session. When disabled, master password must be entered every time the extension popup is opened.

Key Derivation

When generating passwords, this uses a key derived from the master password rather than the master password itself - strenghtening it against attempts to deduce the master password from a brute force attack on a generated password. It does, however, use more CPU time whenever the master password is entered.

Be aware that a large number of iterations may freeze the browser for extended periods of time when master password is entered.

Should be a hexadecimal string - there's no validation of this yet.
